HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab. To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder. A common name for a favicon image is "favicon.ico".

Next, add a <link> element to your "index.html" file, after the <title> element.

Now, save the "index.html" file and reload it in your browser. Your browser tab should now display your favicon image to the left of the page title.

The file format support for a favicon image: ICO, PNG, GIF, JPEG, SVG
